More info:
The slots model versus resources model debate is an ongoing topic of research in the field of psychology. The two models are different hypotheses as to how we store information in our working memory. Working memory is known to have a limit on how much we can store at a time, but it is still unclear how exactly that limit is reached. Many researchers propose that, in reality, our brains use a combination of the slots and resources models to function.

In the slots model, each “item” we want to store fills up a “slot” in our working memory. Our working memory has a small number of these slots, so the amount we can store is limited by how many there are. Note that items always take up the same amount of space in our working memory, as this model proposes that an item is always stored at maximum precision.

In the resources model, items can be stored at different precisions. In other words, we do not always store all available information about the item. Instead, our brain dynamically allocates “resources” for different items based on their precision. In this model, the amount stored in working memory is limited by the total amount of resources available rather than the number of items.
